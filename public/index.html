📖 Panduan Update Database

Langkah update data:

  1. Pilih jenis pegawai (PNS atau CPNS) sesuai file yang akan diupload
  2. Upload file .xlsx hasil export dari SIPDRI
  3. Periksa preview dan statistik data
  4. Klik "Simpan & Update Database"
  5. Download file pegawai.json yang dihasilkan
  6. Upload file tersebut ke GitHub: public/pegawai.json
  7. Tunggu ±30 detik → website otomatis terupdate

Catatan: Data PNS dan CPNS digabung otomatis dalam file pegawai.json. Upload PNS dulu, lalu CPNS — keduanya akan tergabung.
